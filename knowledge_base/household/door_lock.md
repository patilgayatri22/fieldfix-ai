# Door Lock — Repair Guide (Deadbolt / Knob Lock / Latch)

## Common Symptoms
- Key is difficult or stiff to turn in lock
- Key turns but deadbolt or latch doesn't fully extend
- Latch doesn't catch when door closes
- Lock requires lifting or pushing door to turn key
- Key works from outside but not inside (or vice versa)
- Deadbolt won't fully retract with key
- Door lock loose — cylinder or knob wiggles
- Smart lock (electronic) unresponsive

## Likely Causes
- Stiff key → lock cylinder dry or dirty, misaligned door putting stress on cylinder
- Key turns but bolt doesn't extend → internal cam broken, bolt seized from lack of lubrication
- Latch doesn't catch → strike plate misaligned, latch bolt worn or retracted
- Requires door lift to lock → door has sagged, strike plate misaligned with latch
- One-side only works → internal mechanism broken on that side
- Deadbolt won't retract → wrong key (worn), bolt seized, internal gear worn
- Loose cylinder → retaining set screw loose or missing
- Smart lock issues → dead battery (most common), pairing lost, motor fault

## Safe First Checks
- Try key in lock with door open — if it works easily, problem is door alignment not the lock
- Inspect bolt/latch face — should move smoothly when depressed by hand
- Examine strike plate — look for alignment marks where latch is hitting incorrectly
- Check door alignment: gaps around door should be even on all sides
- For smart lock: check battery level first (most have low battery indicator)

## Tools Needed
- Graphite powder or dry PTFE lubricant (for lock cylinder)
- Petroleum jelly or silicone grease (for bolt mechanism)
- Screwdrivers
- Chisel and hammer (for strike plate adjustment)

## Step-by-Step Guidance
1. For stiff key → do NOT use oil-based lubricant (WD-40, etc.) inside a key cylinder — it attracts dirt and makes it worse. Use graphite powder or dry PTFE spray specifically rated for locks.
2. To lubricate cylinder → insert key, apply graphite powder to key blade, insert and remove key 10 times while rotating to distribute lubricant.
3. If key is easier with door open than closed → door is out of alignment, binding the bolt or latch against the strike plate. Tighten door hinge screws first (see door_hinge.md). If still stiff, check strike plate alignment.
4. For strike plate misalignment → close door and observe where latch/bolt contacts the plate. If hitting above or below the hole → strike plate needs to move. Loosen mounting screws, shift plate 1–2mm, tighten. Test.
5. If shifting strike plate slightly is insufficient → enlarge strike plate hole with file or chisel. File metal strike plates, chisel wooden door jamb carefully.
6. For latch that doesn't catch when door closes → check latch front — depressed by hand, it should spring back fully. If sticky → remove latch assembly (2 screws on door edge). Clean and apply silicone grease to latch bolt sides.
7. For lock that requires door lifted to turn → door has sagged (see door_hinge.md). Fix door hinge before addressing lock.
8. For loose lock cylinder → locate set screw (usually accessible from inside face of door). Tighten set screw with correct Allen key or screwdriver.
9. For deadbolt hard to operate → remove deadbolt (4 screws: 2 on interior face, 2 on door edge). Apply silicone grease to bolt sides, internal mechanism. Reinstall.
10. For smart lock unresponsive → replace batteries first. Most smart lock failures are battery-related. If new batteries don't fix it → check app for error codes, or perform factory reset per manual.

## Stop Conditions
- **STOP** if lock cylinder is visibly forced, twisted, or damaged — this may indicate a break-in attempt; assess security before relying on the lock
- **STOP** if deadbolt cylinder can be turned without a key → cylinder is compromised, replace immediately
- **STOP** lubricating with WD-40 inside lock cylinder — causes long-term damage
- Do not attempt to pick or bypass a lock you don't own without authorization

## Prevention
- Lubricate all door locks with graphite or dry PTFE spray annually
- Check strike plate alignment seasonally — doors shift with temperature and humidity
- Tighten all lock screws annually — vibration and use works them loose
- Replace lock batteries (smart locks) annually even if not showing low battery
- Keep a spare key with a trusted neighbor — avoid locksmith call-outs
