# Light Fixture — Repair Guide (Bulbs / Switches / Ceiling Fixtures)

## Common Symptoms
- Light bulb burns out frequently or repeatedly
- Light flickers intermittently
- Light switch makes crackling sound when operated
- Light works in some switch positions but not others
- Room light doesn't work but outlet on same circuit does
- Ceiling fan light kit doesn't work but fan motor does
- Dimmer switch causes humming or buzzing
- Light works briefly after switch is on then goes out

## Likely Causes
- Frequent burnout → voltage too high, vibration loosening filament (incandescent), wrong wattage in enclosed fixture
- Flickering → loose bulb, loose wire connection, failing LED driver, flickering from incompatible dimmer
- Crackling switch → switch contacts arcing, worn switch mechanism — replace switch
- 3-way switch issues → one of the two switches failed, traveler wire loose
- Breaker trips on this circuit → overloaded circuit, short circuit in wiring or fixture
- Fan light not working → separate capacitor for light failed, light kit wiring fault, separate switch not on
- Dimmer hum → dimmer not compatible with LED bulbs — replace with LED-compatible dimmer
- Works briefly then off → thermal protection in LED bulb triggered (enclosed fixture too hot), driver overheating

## Safe First Checks
- Turn off the light switch before touching any bulb
- Check breaker — confirm circuit is not tripped
- Try a different known-good bulb in the socket
- For flickering → tighten bulb (screw in a quarter turn more)
- For dimmer issues → check if bulb packaging says "dimmable"
- For enclosed fixtures → check maximum wattage sticker inside fixture and don't exceed it

## Tools Needed
- Replacement bulbs (correct type, base type, wattage)
- Non-contact voltage tester (highly recommended before any wiring work)
- Ladder or step stool
- Flathead and Phillips screwdrivers

## Step-by-Step Guidance
1. If bulb doesn't work → try a different bulb. Bulbs fail — this resolves most issues.
2. For flickering → tighten the bulb. Clean socket contacts if corroded. If LED bulb flickers → try a different brand; some LEDs flicker with certain fixtures.
3. If bulb works but very dim → check if circuit is on a dimmer set low. Check if dimmable bulb is installed (non-dimmable LEDs buzz and flicker on dimmers).
4. For dimmer hum or buzz → LED bulbs require LED-compatible dimmers. Replace dimmer with one rated for LED loads. Check dimmer packaging specifies LED compatibility.
5. For bulb that overheats in enclosed fixture → enclosed fixtures need enclosed-rated bulbs. Standard LED bulbs trap heat and fail early. Replace with bulbs labeled "enclosed fixture rated."
6. For crackling or popping switch → this is a sign of arcing. Turn off circuit breaker for that circuit. Replace switch. Crackling switches are a fire risk if not addressed.
7. For light not working on 3-way switch (two switches control one light) → test each switch location. If one switch location works and other doesn't → the non-working switch has failed. Replace it.
8. For ceiling fan light kit not working → check if separate pull chain or wall switch controls just the light. If pull chain → cycle pull chain. If bypass is set to off, light won't work from wall switch.
9. If circuit breaker trips when light is turned on → do not repeatedly reset breaker. There may be a wiring fault or overloaded circuit. Reduce load on circuit or consult an electrician.
10. For any work beyond bulb replacement → **turn off the circuit breaker** and verify power is off with non-contact voltage tester before touching any wires.

## Stop Conditions
- **STOP** if you see burn marks or melted plastic around a light socket — fire risk, do not use circuit until inspected by electrician
- **STOP** if breaker trips repeatedly — do not keep resetting; find cause first
- **STOP** any wiring work if voltage tester indicates power is still present — verify breaker is off
- **STOP** if you are not comfortable with any electrical work — call a licensed electrician
- Do not exceed fixture's maximum wattage rating — fire risk from overheating

## Prevention
- Use LED bulbs — they last 15–25× longer than incandescent and run much cooler
- Use enclosed-fixture-rated LEDs in enclosed fixtures — prevents early failure
- Test all circuit breakers and light switches annually — crackling or slow-to-trip breakers are signs of wear
- Never daisy-chain extension cords or power strips to increase fixture count on one circuit
