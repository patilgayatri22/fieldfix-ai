# Tool Safety — Guide (Hand Tools / Power Tools / Workspace)

## Common Symptoms (of unsafe conditions)
- Tool slips during use — screwdriver rounds out screw, wrench slips off nut
- Power tool makes unusual noise or vibration
- Drill bit or blade wobbles during operation
- Tool cord is frayed, cracked, or has exposed wires
- Safety guard on power tool is removed or bypassed
- Workspace is cluttered with poor lighting
- Holding workpiece by hand while using power tool
- Using incorrect tool for the task (e.g. wrong screwdriver type)

## Likely Causes
- Tool slipping → wrong tool size/type, worn tool, incorrect technique
- Unusual noise → worn bearing, loose component, blade damaged
- Bit/blade wobble → chuck not fully tightened, damaged arbor, runout in drill
- Frayed cord → mechanical damage, age, sharp edge cut
- Removed guard → bypassed for convenience — extremely dangerous
- Cluttered workspace → items in reach of moving tools, no clear path
- Hand-holding workpiece → hands in tool's path when slip occurs
- Wrong tool → forces increased, control reduced

## Safe First Checks
- Inspect tool cord before every use — any crack, fray, or exposed wire = do not use until repaired
- Check that all guards and safety features are installed and functional
- Ensure drill chuck, saw blade, or bit is fully tightened before starting
- Verify workpiece is clamped or secured — never hand-hold when using drill, saw, or grinder
- Check workspace lighting — poor visibility leads to mistakes
- Wear appropriate PPE: safety glasses minimum, hearing protection for prolonged power tool use

## Tools Needed
- Safety glasses (required for all power tool work)
- Work gloves (for sharp materials handling — not for rotating power tools)
- Hearing protection (for tools above 85dB — most power tools)
- Clamps or vise (workholding)
- Correct-sized screwdrivers and bits for fasteners being worked

## Step-by-Step Guidance
1. Match tool to fastener/task: Phillips screwdriver on Phillips screws, correctly-sized wrench on hex nuts. Mismatched tools slip and cause injury.
2. Inspect tool condition before use: any cracks in handle, worn tip, or damaged blade = replace before use.
3. For power tools → inspect cord from plug to tool for any damage. Taped-over cuts are not repaired cords. Replace cord or retire tool.
4. Secure workpiece before drilling, cutting, or grinding. Use a clamp, vise, or bench stop. Hands should be minimum 10cm from any cutting zone.
5. For drilling → start drill at low speed, then increase. Let drill do the work — do not force. Force = bit grab and rotation of workpiece.
6. For screwdrivers → apply firm inward pressure while turning. Screw should not strip if pressure is applied correctly. Stripped = wrong bit, too little pressure, or too much torque.
7. Keep cutting tools sharp — dull tools require more force, slip more easily, and cause more injuries than sharp tools.
8. Disconnect power tool from power before changing bits, blades, or accessories. Wait for moving parts to fully stop.
9. Store sharp tools (chisels, utility knives, drill bits) with covers or in roll-up tool wraps — not loose in drawers where hands search blind.
10. After any repair task → account for all tools before closing up a wall, cabinet, or appliance. A tool left inside a device causes damage later.

## Stop Conditions
- **STOP** using any power tool with a damaged cord — electrical shock and fire risk
- **STOP** if power tool makes grinding, squealing, or loud new noise — shut off, inspect before continuing
- **STOP** if drill bit, blade, or disc shows visible damage (chipped, bent, cracked) — flying debris risk
- **STOP** if workpiece shifts during cutting or drilling — secure before continuing
- **Never** remove or bypass safety guards on power tools — they exist to prevent serious injury
- **Never** wear gloves while using rotating power tools (drill, lathe, grinder) — gloves can catch and pull hand in
- **Never** operate power tools while fatigued, distracted, or impaired

## Prevention
- Inspect tools monthly for wear, damage, or cord fraying
- Keep cutting edges sharp — a sharp tool requires less force and is safer
- Always clamp or secure workpieces before using any powered tool
- Store tools in organized toolbox or wall mount — not loose where tips can injure hands
- Take a basic tool safety course if new to power tools
