# Caveman Guide for FieldFix AI

Caveman is a Claude Code helper that compresses responses so Claude gives shorter, more direct implementation output.

Use Caveman for:
- Small coding tasks
- Bug fixes
- File edits
- Refactors
- Import fixes
- Type/schema alignment
- Commit message generation
- Quick code review

Do NOT use Caveman for:
- Safety policy design
- Architecture planning
- README writing
- Pitch writing
- Demo narration
- Product strategy
- Repair-agent prompt design
- Any task where clarity is more important than brevity

---

claude plugin marketplace add JuliusBrussee/caveman
